FRHD Lang is a simple language to generate tracks.
It's indentation based, meaning you need to increase the amount of spaces you add before a line inside a statement.
There must be a new line after each statement.
The playground has presets for more advanced examples. 


var
Statement
This statement declares a variable under the given name, which can be used later in code. Some built in variables: 
x
, 
y
, 
random(a, b)

Usage: 
var <name> <expression>

<name>
can be any text starting with a letter or an underscore, and only including numbers, letters, or underscores later. 
<expression>
can be a number, variable, or a function call.
Example: 
var someName 2000


increment
Statement
This statement increases a variable by a given amount.
Usage: 
increment <name> <expression>

<name>
can be any text starting with a letter or an underscore, and only including numbers, letters, or underscores later. 
<expression>
can be a number, variable, or a function call.
Example: 
increment someName 2000


decrement
Statement
This statement decreases a variable by a given amount.
Usage: 
decrement <name> <expression>

<name>
can be any text starting with a letter or an underscore, and only including numbers, letters, or underscores later. 
<expression>
can be a number, variable, or a function call.
Example: 
decrement someName 2000


physics
Statement
This statement adds a physics line to the track.
Usage: 
physics <x> <y> <x2> <y2>

All arguments can be any expression evaluating to a number.
Example: 
physics x y x+2 y+2


scenery
Statement
This statement adds a scenery line to the track.
Usage: 
scenery <x> <y> <x2> <y2>

All arguments can be any expression evaluating to a number.
Example: 
scenery x y x+2 y+2

repeat
Statement
This statement repeats the indentation block a given amount of times.
Usage: 
repeat <expression>

<expression>
can be a number, variable, or a function call.
Example: 
repeat 100
  var example 42
Notice how it requires indentation. (spaces before 
var example 42
) 

if
Statement
This statement executes the indentation block if the given condition is true or truthy.
Usage: 
if <condition>

<condition>
can be any expression.
Example: 
if x > 100
  var example 42
Notice how it requires indentation. (spaces before 
var example 42
) 

while
Statement
This statement executes the indentation block while the given condition is true or truthy.
Usage: 
while <condition>

<condition>
can be any expression.
Example: 
while x > 100
  var example 42
Notice how it requires indentation. (spaces before 
var example 42
) 

javascript
Statement
This statement executes the given indentation block in JavaScript.
Example: 
javascript
  alert('Cool, huh?');
