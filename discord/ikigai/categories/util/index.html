Util Commands

Utility commands.

Commands

i!activities

View all activities of a user.

i!addemoji

Add an external emoji to the server.

i!covid

Get worldwide or country-based COVID-19 stats.

i!emoji

Get info and the image of an emoji.

i!info

Get information about the bot.

i!invite

Get information about an invite.

i!lyrics

Search the lyrics of a song.

i!math

Calculate a mathematical expression.

i!osu

Get data of an osu! account.

i!poll

Create a poll.

i!user

Get info about a user.
