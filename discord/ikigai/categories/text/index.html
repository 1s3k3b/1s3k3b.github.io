Text Commands

Text related commands.

Commands

i!asciitext

Convert text to ASCII art.

i!base64

Conversions between utf8 and base64.

i!binary

Conversions between utf8 and binary.

i!cancer

Build a text from fancy unicode characters.

i!elements

Build text up from elements of the periodic table.

i!emojify

Replace letters and optionally numbers with emojis in a text.

i!phonetic

Convert text to IPA.

i!pop

Send a text in spoilers multiple times.

i!roman

Conversions between roman and arabic numerals.

i!shrek

Replace some words in the "Shrek is love, shrek is live" copypasta.

i!triangle

Build a triangle from text or emojis.

i!zws

Get a zero-width-space (U200B) character.
