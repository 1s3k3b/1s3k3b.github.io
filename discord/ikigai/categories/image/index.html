Image Commands

Image manipulation commands.

Commands

i!deepfry

Deepfry an image.

i!invert

Invert an image.

i!pixelate

Apply japanese censorship laws to an entire image.

i!qrcode

Read a QR code.

i!thispersondoesnotexist

Get an AI-generated person from [tpdne](https://thispersondoesnotexist.com)
