i!hanime search

Category: Hanime

Search videos. All flags are optional.

NSFW

Examples

i!hanime search aibeya

i!hanime search tenioha --orderby=likes

i!hanime search --tags="blow job,boob job"

Aliases

find

Args

<term>: The term to search for.

Flags

--tags: The tags to search for, seperated by commas.

--tagMode: The tag mode to search with (and/or).

--brands: The brands to search for.

--orderBy: What to order videos by. (created/views/likes/released/title)
