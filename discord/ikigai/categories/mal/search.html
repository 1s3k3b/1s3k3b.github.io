i!mal search

Category: MyAnimeList

Search anime or manga.

Examples

i!mal search kitsune

i!mal search kimi --rated=Rx

i!mal search --genre=isekai

Aliases

find

Args

[type]: The type of data to search for. Defaults to anime.

[term]: The term to search for.

Flags

--genre: The optional genre to find.

--rated: The optional rating to find: `g`, `pg`, `pg13`, `r17`, `r`, `r`, `rx`
